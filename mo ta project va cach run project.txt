Project này là là giả lập api với json server, để get các product từ server về
Trong đây sẽ có chức năng thêm sản phẩm, các tính năng của giỏ hàng như thêm xóa cập nhật giỏ hàng và lưu vào localStorage
Cách chạy project
b1 : mở cmd và nhập : json-server --watch db.json
b2 : mở cmd khác và chạy project : npm start
